Share links
When a customer shares a campaign, a share link is created.

Metadata

:share_link:
- id
- campaign_id
- customer_id
- code
- created_at
>>>

GET /api/:version/share_links/:id

Returns the campaign via the share link (code) provided

Examples

{
  response => Hash {Campaign Data}
}

Params

Param name Description
id
required

The share link's ID (code)


Value:

Must be String
>>>

POST /api/:version/share_links

Create a new share link for a campaign and return the URL

Errors

422 Invalid parameters. Link was not created

Examples

{
  "response" => Hash{
    "url" => #the complete URL of the share link
  }
}

Params

Param name Description
share_link
required
Value:

Must be a Hash
share_link[campaign_id]
required

The ID of the campaign being shared


Value:

Must be a number.
